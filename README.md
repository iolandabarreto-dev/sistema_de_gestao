## 📌 Sistema Interativo de Gestão de Turmas


## 🎯 Objetivo

O **Sistema Interativo de Gestão de Turmas** foi desenvolvido para organizar, gerenciar e visualizar informações de turmas de forma dinâmica e prática. Este projeto foi realizado como parte do **curso do programa CaixaVerso - Dev Frontend Nível 1 da ADATECH**, com foco em aplicar conceitos de JavaScript e manipulação de DOM.

📚 Este projeto foi proposto pela AdaTech, como parte do programa CaixaVerso — Desenvolvimento Front-End Nível 1.

---

## ✨ Funcionalidades

* Adicionar novas turmas e alunos de forma interativa.
* Visualizar a lista de turmas cadastradas.
* Remover turmas ou alunos quando necessário.
* Interface simples e intuitiva para fácil navegação.
* Integração entre HTML e JavaScript para atualização dinâmica da página.

---

## 🛠️ Tecnologias utilizadas

* **HTML5** – Estrutura do projeto.
* **JavaScript** – Lógica do sistema e manipulação de dados.
* **CSS (opcional)** – Estilização básica para interface amigável.
